# Contributing to Nordic Stage

1. **Pick an issue:** Check the GitHub project board and assign yourself an issue.
2. **Branching:** Create a feature branch: `feature/{track}-{milestone}-{short-name}`.
3. **Local Dev:** Use `bun run dev:web` and `uv run python manage.py runserver`.
4. **Pull Requests:** Open a PR against `main`. All CI checks must pass.
